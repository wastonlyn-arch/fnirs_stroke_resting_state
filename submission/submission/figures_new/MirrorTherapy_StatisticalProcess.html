1. Data Pre-processing / 数据预处理

1.1 Functional Connectivity / 功能连接计算

中文：每个被试的静息态 fNIRS 数据（0.01–0.1 Hz 带通滤波后）中，计算所有 38 个通道两两之间的 Pearson 相关系数 $r$，然后通过 Fisher Z 变换转换为近似正态分布的 Z 值：

Z = arctanh(r)

1.2 Channel Selection / 通道选择

对每个被试分别处理基线（ses-baseline）和后测（ses-post）两个时间点。每个时间点的 .mat 文件包含 Z（连接矩阵）和 good_channels（通过质量控制的通道索引）。对于每个被试：

  • 取两个时间点 good_channels 的交集作为共同好通道：common = intersect(good_pre, good_post)
  • 仅用共同好通道计算所有边（上三角元素）
  • 若某条边的任一端点不在 common 中，该边标记为 NaN
  • 剔除标准：若共同好通道数 < 总通道数的 30%（即 < 12/38），则剔除该被试

For each subject, Pearson correlation r between all 38 channel pairs is computed from resting-state fNIRS data (bandpass 0.01–0.1 Hz), then Fisher Z-transformed. Pre and post sessions each have a .mat file with Z matrix and good_channels. The intersection of pre and post good channels is used. A subject is excluded if fewer than 30% of channels (12/38) are common.

1.3 Delta Z Calculation / ΔZ 计算

对每条边 $(i,j)$，被试 $s$ 的 ΔZ 为：

delta_Z[i,j] = Z_post[i,j] - Z_pre[i,j]

对于组件级分析（SNB_deltaZ、ISS_deltaZ），组件的平均 ΔZ 为：

mean_delta_Z_s = mean(delta_Z[i,j] for all edges (i,j) in the component)

注意：仅对非 NaN 的边求平均，不同被试可能基于不同数量的有效边。

For each edge (i,j), delta_Z = Z_post - Z_pre. For component-level analyses, the mean delta_Z is the average over all non-NaN edges within the component.

2. Network-Based Statistics (NBS) / 网络统计

2.1 SNB_NBS: Interaction Effect NBS / 交互效应 NBS

Model / 模型：

Z_post[i,j] ~ 1 + MT + PG + MT:PG + Z_pre[i,j]

对每条边 $(i,j)$ 独立拟合上述 ANCOVA 模型（OLS + Type III SS），提取交互项 MT:PG 的 t 统计量。

For each edge (i,j), an ANCOVA model is fitted independently: Post_FC ~ MT + PG + MT:PG + Pre_FC. The t-statistic for the MT:PG interaction is extracted.

2.3 P-value Calculation / p 值计算

对每条观测到的组件，p 值为：

p = (sum(max_sizes_null >= n_edges_observed) + 1) / (n_permutations + 1)

其中 max_sizes_null 是 5000 次置换中每次最大组件大小的向量。加 1 为避免 p = 0。

The p-value for each observed component is computed from the permutation null distribution: (count of null max sizes >= observed size + 1) / (5000 + 1).

3. ANOVA / ANCOVA Models / 方差与协方差分析

3.1 Component-Level ANOVA (SNB_deltaZ, ISS_deltaZ)

delta_Z_component ~ 1 + MT + PG + MT:PG (Type III SS)

中文：在已确定的 NBS 组件上，以组件平均 ΔZ 为因变量，进行两因素完全析因 ANOVA。选择 Type III 平方和的原因：

  • 适用于存在交互项的析因设计
  • 对不平衡设计（n=6 vs n=8）更稳健
  • 是交互效应检验的标准选择（Maxwell & Delaney, 2004）

Two-way factorial ANOVA on the component mean delta_Z. Type III SS is chosen because it is standard for factorial designs with interaction, robust to unbalanced samples (n=6 vs n=8), and expected by reviewers.

3.2 Post-hoc ANCOVA (Subnetwork Analysis)

Post_FC_subnetwork ~ 1 + MT + PG + MT:PG + Pre_FC_subnetwork (Type II SS)

中文：在子网络分解分析中，采用 Type II 平方和。原因：在 ANCOVA 中，Type II 对主效应的检验在协变量存在时比 Type III 更稳定（Fox, 2016），且交互项 F 值在 Type II 和 Type III 下一致。

Type II SS is used for ANCOVA with covariates because it avoids non-essential collinearity issues (Fox, 2016). The interaction F-value is identical between Type II and Type III.

3.3 CATE Model / CATE 模型

Post_FC ~ 1 + Group + Pre_FC (Type II SS)

用于计算组效应，作为补充参考，不进行交互推断。

Group effect model for supplementary reference. No interaction inference is drawn.

4. Bootstrap Confidence Intervals / Bootstrap 置信区间

4.1 Parameters / 参数

ParameterValue
Method / 方法Non-parametric bootstrap (Efron, 1979)
Resampling unit / 重采样单位Subject (with replacement)
Iterations / 迭代次数1000
CI method / CI 方法Percentile
CI level / 置信水平90% (5th & 95th percentiles)
Random seed / 随机种子20240507

4.2 Algorithm / 算法

对每个要 Bootstrap 的统计量 $\theta$（如偏 $\eta^2$、协同指数、$\beta$ 系数）：

  1. 从原始 $N$ 个被试中有放回抽样 $N$ 次 → 一条 Bootstrap 样本
  2. 在该样本上重新计算 $\theta$
  3. 重复 1000 次 → $\theta^*_1, \theta^*_2, \dots, \theta^*_{1000}$
  4. 剔除失败的样本（如某次 Bootstrap 中某组缺失导致模型不可拟合）
  5. 取 $B_{\text{valid}}$ 个有效样本的 5% 和 95% 百分位数作为 90% CI

Non-parametric bootstrap: resample subjects with replacement, recompute statistic, repeat 1000 times. Failed samples (e.g., empty group) are excluded. Percentile method for 90% CI.

4.3 Bootstrap for Synergy Index / 协同指数 Bootstrap

协同指数定义为：

Synergy = mu[MtPg] - mu[MT] - mu[PG] + mu[sham]

每次 Bootstrap 中，对有放回抽得的被试按组求均值，计算协同指数。若某次 Bootstrap 中某组无被试，该次记为 NaN 并丢弃。

Synergy = MtPg mean - MT mean - PG mean + sham mean. In each bootstrap iteration, group means are computed from the resampled subjects. If a group is empty, the iteration is discarded.

在执行任何推断性统计检验之前，自动运行以下前提假设检验：

5.2 Output Format / 输出格式

>>> [R10 Assumption Audit] > - Normality (Shapiro-Wilk): W=0.99, p=0.95 > - Homogeneity (Levene): W=1.05, p=0.39 > - Slope Homogeneity (ANCOVA only): p=0.12 > - Status: [PASSED]

Each statistical test is preceded by an automated assumption audit block. If assumptions are violated, a warning is issued and an alternative method is recommended.

5.3 Decision Logic / 决策逻辑

  • PASSED ($p > 0.05$): Report parametric results directly
  • WARNING ($p < 0.05$): Issue warning, suggest data transformation, Welch correction, or non-parametric alternative
  • In this project, the ANCOVA slope homogeneity failed (p=0.042), so the main analysis switched to $\Delta Z$ ANOVA which does not rely on this assumption

6. Multiple Comparison Correction / 多重比较校正

6.1 NBS Level / NBS 层级

NBS 本身在组件级（而非边级）控制 FWER。通过 5000 次置换检验，每次记录最大组件大小，观测组件的大小在零分布中的位置决定其显著性。校正对象是组件，而非逐条边。

NBS controls FWER at the component level (not edge level). The 5000-permutation null distribution of max component size determines significance. The correction is for the number of components, not edges.

6.2 Post-hoc Level / 事后比较层级

对 CATE 的两两比较，采用 FDR-BH 校正（Benjamini & Hochberg, 1995），通过 statsmodels 的 multipletests(method='fdr_bh') 实现。

6.3 Brain-Behavior Level / 脑-行为层级

24 对 Spearman 相关全部经过 FDR-BH 校正。仅当 $q < 0.05$ 时才认定为统计显著。本项目中无相关对通过校正。

All 24 brain-behavior Spearman correlations are FDR-BH corrected. None survived correction in this project.

6.4 Subnetwork Level / 子网络层级

子网络分解分析（Inter-Hemi, Ipsi-Intra, Contra-Intra）明确标注为探索性、未校正，不用于确证性推断。

Subnetwork decomposition is marked as exploratory and uncorrected, not for confirmatory inference.

7. Effect Size Calculation / 效应量计算

7.1 Partial Eta-Squared / 偏 η²

eta2_p = SS_effect / (SS_effect + SS_error)

适用于所有 ANOVA/ANCOVA 模型。以 Bootstrap 90% CI 作为不确定性度量。

Partial eta-squared for all ANOVA/ANCOVA models, with bootstrap 90% CI.

7.2 Cohen's d / Cohen d

d = (mean_1 - mean_2) / pooled_sd

用于 CATE 模型的两两比较。

Cohen's d for pairwise comparisons in the CATE model.

7.3 Spearman's Rho / Spearman 等级相关

rs = 1 - 6 * sum(d_i^2) / (n * (n^2 - 1))

用于脑-行为相关性分析（选择 Spearman 而非 Pearson 的原因：部分临床变量分布非正态）。

Spearman rank correlation for brain-behavior analysis (chosen over Pearson due to non-normal clinical variables).

7.4 Synergy Index / 协同指数

Synergy = mu(MtPg) - mu(MT) - mu(PG) + mu(sham)

等价于 2×2 析因设计中交互项的 $\beta$ 系数。正值表示超加性协同（1+1 > 2），负值表示拮抗/网络精简（1+1 < 2）。

The synergy index equals the interaction coefficient beta in a 2x2 factorial design. Positive = super-additive synergy; negative = antagonistic / network pruning.
