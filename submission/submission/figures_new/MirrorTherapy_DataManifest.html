文件说明

根目录（报告）

文件说明格式
MirrorTherapy_FinalReport主分析报告（MathJax），包含：SNB_NBS 网络发现、SNB_deltaZ 效应量化、ISS_deltaZ 敏感性分析、交叉验证。HTML + PDF
MirrorTherapy_SupplementaryReport补充材料，包含：ISS 统计描述、CATE 组效应、脑-行为散点图、子网络分解。HTML + PDF
MirrorTherapy_StatisticalProcess统计分析过程说明（中英双语），含所有方法的数学公式和参数选择理由。HTML + PDF

main/ (SNB_NBS + SNB_deltaZ)

文件说明格式
SNB_deltaZ_ANOVA_Table.csvSNB 组件 ΔZ ~ MT×PG 的 Type III ANOVA 结果：F 值、p 值、偏 η² 及 90% CI。CSV
SNB_deltaZ_Synergy.csvSNB 组件的协同指数及 90% CI，含四组均值。CSV
SNB_deltaZ_Beta.csv交互项回归系数 β 及 90% CI（Bootstrap）。CSV
SNB_deltaZ_AssumptionChecks.csvR10 假设审计：Shapiro-Wilk 正态性、Levene 方差齐性检验结果。CSV
SNB_deltaZ_SubjectData.csv每个被试的 SNB 组件 ΔZ、Pre_FC、Post_FC、组标签。CSV
SNB_deltaZ_ComponentSummary.csvSNB 组件摘要：边数、节点数、阈值、p 值。CSV
SNB_deltaZ_Metadata.txt完整元数据：随机种子、Bootstrap 参数、样本量、模型公式。TXT

main/figures/

文件说明格式
SNB_NBS_Heatmap38×38 t 值矩阵热力图，12-ROI 排序，SNB 边用黑框标出。PNG + PDF
SNB_NBS_Topoplot2D 头皮拓扑图：节点大小=度中心性，颜色=功能分区，边色=t 值。PNG + PDF
SNB_NBS_3DGlass4 视角玻璃脑图（左侧、右侧、背侧、腹侧），基于 nilearn。PNG + PDF
SNB_NBS_3D_interactive交互式 3D 连接组 HTML（nilearn view_connectome）。HTML
SNB_NBS_NullDistNBS 零分布直方图：观测组件大小（红线）vs 5000 次置换分布。PNG + PDF
SNB_deltaZ_BoxplotSNB 组件 ΔZ 箱线图，叠加个体散点。PNG + PDF
SNB_deltaZ_InteractionMT × PG 交互作用图（单元格均值 + 标准误）。PNG + PDF
SNB_deltaZ_Synergy协同示意图：四组均值条图 + 加性预测虚线。PNG + PDF

sensitivity/ (ISS_deltaZ 敏感性分析)

文件说明格式
ISS_deltaZ_ANOVA_Table.csvISS 组件 ΔZ ~ MT×PG 的 Type III ANOVA 结果。CSV
ISS_deltaZ_Synergy.csvISS 组件的协同指数及 90% CI。CSV
ISS_deltaZ_AssumptionChecks.csvISS 组件分析的 R10 假设审计。CSV
ISS_deltaZ_SubjectData.csv每个被试的 ISS 组件 ΔZ 和组标签。CSV
ISS_deltaZ_Summary.csvISS 组件摘要：边数、节点数、p 值。CSV

sensitivity/figures/

文件说明格式
ISS_deltaZ_BoxplotISS 组件 ΔZ 箱线图（敏感性分析）。PNG + PDF
ISS_deltaZ_InteractionISS 组件 MT × PG 交互作用图（敏感性分析）。PNG + PDF

supplementary/（补充材料）

文件说明格式
BrainBehavior_Correlation.csvSpearman 相关表（24 对）：rₛ、原始 p 值、FDR q 值、Bootstrap 95% CI。CSV
BrainBehavior_*_vs_*.png/pdf24 张散点图：6 个脑指标 × 4 个临床指标，按组着色，含 Spearman rₛ 和回归线。PNG + PDF

命名规范

格式： MirrorTherapy_{分析名}_{指标}_{日期}.ext
分析类型： SNB_NBS（网络发现）、SNB_deltaZ（效应量化）、ISS_deltaZ（敏感性）、BrainBehavior（脑-行为相关）
日期格式： YYYY-MM-DD
可用格式： PNG（快速预览）、PDF（打印/发表）、CSV（数据）、TXT（元数据）、HTML（交互式）
