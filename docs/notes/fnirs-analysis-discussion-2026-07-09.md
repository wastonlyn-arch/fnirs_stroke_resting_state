## 1. 生理基础：fNIRS 究竟在测什么？

fNIRS 的核心原理是**神经-血管耦合（neurovascular coupling）**——神经元活动增加 → 局部脑血流（CBF）增加 → 氧合血红蛋白（HbO）浓度上升、脱氧血红蛋白（HbR）浓度下降。因为 HbO 和 HbR 在近红外波段（650-900nm）有显著不同的吸收光谱，用两个波长的光穿过组织后的衰减量就能反推出两种色团的浓度变化。

`fnirs_loader.py:138-172` 实现的 MBLL（Modified Beer-Lambert Law）公式：

$$
\Delta OD(\lambda) = -\log\left(\frac{I(t)}{I_0}\right) = \varepsilon(\lambda) \cdot \Delta C \cdot d \cdot DPF(\lambda)
$$

#### 2.2.1 零相位滤波 `filtfilt`（原理讨论 2026-07-09）

带通滤波（频域，滤什么）和 `filtfilt`（时域，怎么滤）是两层独立概念。

- **`lfilter`**：前向卷积，信号峰值被延迟。对 fNIRS 来说 HRF 的峰值时间会被移动 → GLM 的 design matrix 和实际信号错位 → beta 估计偏小
- **`filtfilt`**：前向滤波 → 翻转 → 反向再滤一遍 → 翻转回来。两次滤波的相移互相抵消，峰值位置不变，零相位失真。代价是有效阶数翻倍（3 阶 → 6 阶衰减）
- 边沿效应：信号两端滤波质量差，但基线窗口和末端静息段可容忍

代码中 `butter`（设计）和 `filtfilt`（施加）是独立的两个步骤，同一个 b, a 用 `lfilter` 也能跑但会偏移。

#### 2.2.2 多项式漂移建模（原理讨论 2026-07-09）

GLM 设计矩阵中的多项式项 (`t⁰, t¹, t², t³`) 建模慢漂移：

$$y(t) = \beta_{\text{task}} X_{\text{task}} + \beta_1 + \beta_2 t + \beta_3 t^2 + \beta_4 t^3 + \varepsilon$$

| 项 | 建模什么 |
|---|---|
| β₁·1 | 常数偏移（DC 水平） |
| β₂·t | 线性漂移 |
| β₃·t² | U 形/倒 U 形漂移 |
| β₄·t³ | 复杂弯曲漂移 |

为什么 t 要 z-score 标准化：t³ 在 1000 秒量级是 10⁹，和 task regressor（量级 ~1）同矩阵求逆会数值不稳定。标准化后均值为 0、std=1，t³ 在个位数。

3 阶是 fNIRS 标准选择 — 能去除大多数慢漂移形状，不浪费自由度。与带通滤波互补：滤波去掉 <0.01 Hz，多项式兜底残余漂移。

#### 2.3.1 PCA vs ICA 空域去噪（原理讨论 2026-07-09）

**为什么需要空域分解？**

每个通道测到的不是纯脑信号，而是多源混合：

```
通道 5 的信号 = 脑信号(左侧M1)×0.3 + 脑信号(前运动区)×0.1
              + 头皮血流×0.4 + 呼吸×0.15 + 随机噪声×0.05
```

空域滤波器利用 38 通道之间的空间关系来分离这些源。

**PCA（Pearson, 1901）**

- 原始动机：教育测量 — 10 门课分数背后只有 3 个"真正能力"
- 几何直觉：38 维点云在哪个方向上"拉得最长" → PC1 = 方差最大的方向
- fNIRS 中：全局血压波动让 38ch 同步涨落 → PC1 直接抓到。丢前 1-3 PCs = 去全局噪声

**ICA（Jutten & Hérault, 1991）**

- 原始动机：鸡尾酒会问题 — 3 个麦克风 + 3 个人同时说话，怎么分离
- 核心洞察：真实物理源是非高斯的（语音有静音/爆破音，心跳有尖峰/平段）
- fNIRS 中：心跳/呼吸作为独立周期源被分离出来，人工挑出噪声 IC 并去掉

**决策：什么时候用哪个？**

| 噪声类型 | 空间特征 | 该用谁 |
|------|------|------|
| 血压/Mayer波 (~0.1 Hz) | 全脑 38ch 同步 | **PCA**（PC1 直接抓到） |
| 心跳 (~1 Hz) | 前额强，后部弱 | **ICA**（独立周期源） |
| 头动伪影 | 相邻 3-5ch 同步跳 | **小波**（你已有）> PCA |
| 任务诱发脑信号 | 局部（M1 附近 3-5ch） | **两者都不能去！** |

**核心风险**：PCA 和 ICA 都是盲源分离，不知道什么是噪声、什么是脑信号。

**对你的具体建议：**

| 场景 | 推荐 | 理由 |
|------|------|------|
| 静息态 FC | PCA 去掉前 1-3 PCs | 全局血压波人为抬高 FC（假正连接），审稿人接受度较高 |
| 任务态 GLM | 不做，或仅 PCA 去 PC1 | GLM 把非锁时噪声归入残差，额外去噪可能吃掉激活信号 |
| 质量控制 | 两者都跑，画成分图 | 确认小波校正效果，不一定要去掉 |
| 无短距通道时 defend | PCA + 引用文献 | 比什么都不做强，比 GSR 争议小 |

**在管线中的位置**：`MBLL → [PCA/ICA] → BPF → GLM/FC`（去噪在 HbO/HbR 转换后，滤波前）

#### 3.2d.1 混合效应模型 vs 置换检验（原理讨论 2026-07-09）

两种不同的统计推断路线，不是同一东西的两种实现：

**A. 参数方法 — statsmodels 混合效应模型**
- 假设：beta 服从正态分布，残差 i.i.d.
- 方法：ML/REML 估计固定效应 + 随机效应方差分量
- 输出：β、SE、t 值、p 值（标准线性混合模型输出）
- 优点：审稿人熟悉（= R `lme4`），处理重复测量 + 缺失数据，速度快
- 缺点：数据不满足正态假设时 p 值不可靠
- 代码：`smf.mixedlm("beta ~ Group * Time", groups="Subject", re_formula="1+Time")`

**B. 非参数方法 — 置换检验**
- 假设：无分布假设
- 方法：打乱标签 5000 次 → 零分布 → 真实效应在零分布的位置
- 输出：经验 p 值（随机效应超过真实效应的比例）
- 优点：不依赖分布假设，离群值鲁棒，小样本友好，审稿人接受度高（类似 NBS 逻辑）
- 缺点：复杂交互的置换策略需仔细设计（分层重抽样）

**推荐组合方案**（领域最常见）：

| 水平 | 用什么 |
|------|------|
| 1st-level 个体 GLM | OLS（你已有的 `np.linalg.lstsq`） |
| 2nd-level 组分析 | **statsmodels 混合效应模型**（`beta ~ Group * Time + (1|Subject)`） |
| 多重比较校正 | **置换检验**（cluster-based permutation 或 FDR） |
| 最终报告 | 混合效应模型的 β + 置换校正后的 p 值 |

升级路径：pre-post 减法 → `smf.mixedlm("beta_task ~ Group * Time", groups="Subject")` → 对 38ch 交互项 p 值做 FDR

## 5. 前沿与方法学争议

### 5.1 HbO vs HbR 的解读
- HbO：SNR 高，但可能包含更多系统性生理噪声
- HbR：空间特异性更好，部分效应只在 HbR 上出现
- 推荐策略：同时 report 两者，不做选择性报告
Q: 现有的管线只报了 HbO，其他研究也是。是不是得查阅论文确认是否需要同时报告 HbR？只报 HbO 审稿人会质疑吗？
### 5.2 全局信号回归（GSR）
- fMRI 静息态 GSR 是标配
- fNIRS 中争议大：fNIRS 全局信号可能含真实神经活动
- 当前代码没有做 GSR——可能是对的，但需要在论文中说明理由
Q: 为什么要做信号回归？是用短通道、加速度计回归掉伪影吗？GSR 的目的和短距通道回归的区别？

#### 5.2.1 GSR vs 短距通道 vs 加速度计（原理讨论 2026-07-09）

三种常被混淆的方法：

**短距通道回归**（SD < 1.5 cm）：短距通道只穿过头皮/颅骨，测纯系统噪声 → 作为噪声模板回归掉长距通道中的头皮成分。有物理参照，审稿人最喜欢的方案。但你没有这个硬件。

**加速度计回归**：记录头部平移/旋转 → 作为 GLM nuisance regressor。只处理头动，不处理血压/呼吸/心跳。通常和小波去噪互补。

**GSR（全局信号回归）**：全局信号 = 38 通道逐时间点均值。`hbo_clean = hbo - β × global_signal`。不需要任何额外硬件，纯数学。

| | 短距通道 | 加速度计 | GSR |
|------|------|------|------|
| 额外硬件 | ✅ 需要 | ✅ 需要 | ❌ 不需要 |
| 去掉什么 | 头皮血流 | 头动 | 所有通道共享的任何东西 |
| 会不会去掉脑信号 | 不会 | 不会 | **会**（如果脑信号也是全局的） |
| 争议 | 无（金标准） | 无 | **有** |

**GSR 争议核心**：GSR 假设"全脑平均=噪声"，但 fNIRS 仅 38ch 覆盖部分皮层，全局信号可能是真实广泛神经活动。卒中后可塑性常涉及双侧广泛激活，GSR 可能正好去掉目标信号。静息态 FC 中去 GSR 后出现的负相关真假仍无定论。

**我们没有短距通道和加速度计，只有三个选择**：
- 不做 SPN 去除 → limitations 中 defend
- PCA 去 PC1 → 和 GSR 高度相关但争议稍小
- GSR → fNIRS 中争议最大，不推荐

**推荐策略**：静息态 PCA 去 PC1，任务态不额外去 SPN，methods 中说明理由。

#### 5.4.1 两层敏感性分析框架（原理讨论 2026-07-09）

敏感性分析回答审稿人的核心问题："你的结果是不是因为选了这组参数才显著的？"

分两个层级，控制的是不同的东西：

```
层级 1: QC/数据准入 — 哪些数据进管线？
  SNR, SCI, 坏段比例, 坏通道数, 被试剔除标准
  → 影响进入管线的数据量和质量

层级 2: 预处理参数 — 进去后怎么处理？
  BPF, IQR, GSR/PCA, FC 方法
  → 影响最终 FC/beta 的数值
```

**层级 1 参数详解**：

| 参数 | Homer3 来源 | 含义 | 敏感性来源 |
|------|------|------|------|
| SNR 阈值 | `hmrR_PreprocessIntensity` | 信号幅值低于噪声底限 → 探头接触不良/头发遮挡 | 阈值差 2 dB 可能决定一个通道的生死 |
| SCI（Scalp Coupling Index） | 通常自算 | 衡量探头-头皮耦合质量 | 头发覆盖区 vs 无毛区差异大；深肤色吸收更多光 |
| 坏段检测阈值 | `hmrMotionArtifact` 的 amp_thresh/std_thresh | 信号跳变 > N×SD → 标记伪影段 | 太严 → 真脑信号被标伪影；太松 → 漏进伪影 |
| 通道剔除标准 | 自定 | 坏通道 > N → 整被试剔除 | N=5 vs N=10 可能差出 2-3 个被试 |
| 伪影校正方法 | `hmrMotionCorrectWavelet` vs `Spline` vs `PCA` | 不同方法对不同类型伪影效果不同 | 切换方法 = 完全不同的去噪路径 |

**层级 2 参数详解**（按重要性排序）：

| 排名 | 参数 | 当前值 | 建议变动范围 | 为什么最重要 |
|------|------|------|------|------|
| 1 | 带通滤波频段 | 0.01–0.1 Hz | 下限 [0.005, 0.02]; 上限 [0.08, 0.12] | 静息态 FC 本质上测的就是这个窗里的同步性，窗一挪结果就变 |
| 2 | 运动校正阈值 | db4, IQR=1.5 | IQR [0.8, 1.0, 1.5, 2.0] | 决定去掉多少"疑似伪影"，直接影响信号幅度 |
| 3 | 全局信号处理 | 不做 | 不做 vs PCA 去 PC1 vs GSR | 改变 FC 矩阵全局偏移，直接影响组间对比 |
| 4 | FC 方法 | Pearson r | Pearson vs Spearman vs 互相关 | Pearson 假设线性，非线性耦合下失效 |
| 5 | HRF 模型（任务态） | SPM 双 Gamma | SPM vs FIR vs 无卷积 | 影响 beta 估计峰值 |

**实际操作建议**（不需要全排列，三组即可）：

```
主分析（当前管线）:
  层1: 原定 QC 阈值
  层2: 小波 IQR=1.5 + BPF 0.01-0.1 + 不做 GSR + Pearson FC

敏感性 A（预处理稳健性）:
  层1: 原定
  层2: 小波 IQR=1.0 + BPF 0.005-0.08 + PCA 去 PC1 + Spearman FC

敏感性 B（数据准入稳健性 — 更严格 QC）:
  层1: 提高 SNR/SCI 阈值（少留被试，但数据更干净）
  层2: 原定

敏感性 C（样本流失稳健性 — 更宽松 QC）:
  层1: 降低阈值（多留被试，但允许更多噪声）
  层2: 原定
```

论文中报告：一段话 + supplementary figure（NBS 网络对比图或 beta 热力图对比），引用 Pinti et al. (2019) 和 Yücel et al. (2021) 作为预处理报告规范依据。
